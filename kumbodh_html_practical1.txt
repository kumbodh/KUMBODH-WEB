<h2>1. Creating a GitHub Account</h2>
<ol>
<li>Go to https://github.com</li>
<li>Click on <b>Sign Up</b></li>
<li>Enter email, username, and password</li>
<li>Verify email and complete registration</li>
</ol>
</div>

<div class="section">
<h2>2. Creating a Repository</h2>
<ol>
<li>Login to GitHub</li>
<li>Click the <b>+</b> icon in the top right</li>
<li>Select <b>New Repository</b></li>
<li>Enter repository name</li>
<li>Select Public or Private</li>
<li>Click <b>Create Repository</b></li>
</ol>
</div>

<div class="section">
<h2>3. Uploading Files</h2>
<ol>
<li>Open your repository</li>
<li>Click <b>Add File</b></li>
<li>Select <b>Upload Files</b></li>
<li>Upload your project files</li>
<li>Click <b>Commit Changes</b></li>
</ol>
</div>

<div class="section">
<h2>4. Version Management Using Git</h2>
<p>Basic Git commands used for version control:</p>

<pre>

git remote add origin https://github.com/username/repository.git
git push -u origin main
